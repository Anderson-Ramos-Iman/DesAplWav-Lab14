# Prompt para ejecutar la parte editable del Lab14

```text
Actua como un asistente de desarrollo dentro de este proyecto Next.js y realiza solo cambios que se puedan resolver mediante archivos y codigo del repositorio actual.

Objetivo:
Mejorar el portafolio para cumplir la parte tecnica del Lab14 relacionada con SEO y estructura de paginas.

Tareas que si puedes realizar en este repositorio:
1. Crear una nueva pagina de contacto en `src/app/contact/page.tsx`.
2. Agregar metadata SEO para la pagina de contacto siguiendo el estilo usado en `src/app/layout.tsx` y otras paginas del proyecto.
3. Actualizar `src/app/sitemap.ts` para incluir la ruta `/contact`.
4. Actualizar `src/app/robots.ts` si hace falta para reflejar correctamente el sitemap y las rutas permitidas.
5. Actualizar la navegacion en `src/components/Header.tsx` y, si aporta valor, tambien en `src/components/Footer.tsx` para incluir acceso a la pagina de contacto.
6. Reutilizar `lib/data.ts` para mostrar correo, GitHub, LinkedIn u otra informacion de contacto ya existente.
7. Mantener la estructura actual del proyecto y el estilo general ya presente.

Restricciones:
- No intentes desplegar en Vercel.
- No intentes registrar ni configurar un dominio externo.
- No inventes accesos a plataformas externas.
- Limita los cambios a archivos, rutas, metadata y codigo del proyecto.

Resultado esperado:
- Nueva ruta `/contact` funcional.
- Sitemap actualizado.
- Robots actualizado si corresponde.
- Navegacion consistente con la nueva pagina.
- Codigo claro y coherente con la estructura existente del repositorio.
```
